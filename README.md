#README 

This is a test site in use as a development sandbox to create a site based on Angular.js and Ruby on Rails.

Particulars are:

* Ruby ~2.0.0  
* Rails ~4.0.0  
* MongoDB/Mongoid (database)  
* Angular.js  
* Devise (Authentication)  
* Pundit (Authorization)  
* Twitter Bootstrap (CSS Library)  
* JQuery  
* Bower (Asset management)
